# threejs_portfolio

Abhay Raj's portfolio website built with React + Three.js (React Three Fiber), featuring a scroll-driven 3D storytelling intro and a full portfolio with projects, skills, and contact info.

## Tech Stack
- **Vite** + **React**
- **Three.js** via React Three Fiber + Drei
- **GSAP** for scroll animations
- Premium dark theme with Gruvbox accent colors

## Development
```bash
npm install
npm run dev
```
